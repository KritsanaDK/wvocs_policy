นโยบายการรักษาความมั่นคงปลอดภัยเว็บไซต์

Website Security Policy of The War Veterans Organization of Thailand

องค์การสงเคราะห์ทหารผ่านศึกให้ความสำคัญกับความมั่นคงปลอดภัยของข้อมูลผู้ใช้งาน เพื่อป้องกันการเข้าถึงโดยไม่ได้รับอนุญาต การสูญหาย หรือการถูกทำลายของข้อมูล

1. มาตรการและวิธีการรักษาความมั่นคงปลอดภัย

องค์กรได้ใช้มาตรฐานความปลอดภัยระดับสูง โดยใช้เทคโนโลยี Secure Socket Layer (SSL) ในการเข้ารหัสข้อมูลระดับ 128 บิต เพื่อปกป้องข้อมูลที่ส่งผ่านเครือข่ายอินเทอร์เน็ต ผู้ใช้งานสามารถสังเกตได้จาก URL ที่ขึ้นต้นด้วย https://

2. การใช้ข้อมูลส่วนบุคคลและระบบความปลอดภัย

นอกจากมาตรการพื้นฐาน องค์กรยังใช้เทคโนโลยีเพิ่มเติมเพื่อปกป้องข้อมูลส่วนบุคคลของผู้ใช้ ดังนี้:

  • Firewall: ระบบควบคุมการเข้าถึงข้อมูล อนุญาตเฉพาะผู้มีสิทธิ์เท่านั้น
  • Virus Scan: ติดตั้งโปรแกรมป้องกันไวรัสทั้งในเครื่องผู้ให้บริการและเซิร์ฟเวอร์ พร้อมอัปเดตสม่ำเสมอ
  • Cookies: ใช้เพื่อเพิ่มความสะดวกในการใช้งาน โดยคำนึงถึงความเป็นส่วนตัวของผู้ใช้เป็นหลัก
  • Auto Log Off: ระบบจะออกจากระบบอัตโนมัติเมื่อไม่มีการใช้งาน เพื่อความปลอดภัยของผู้ใช้

3. คำแนะนำด้านความปลอดภัยสำหรับผู้ใช้

แม้ว่าองค์กรจะมีมาตรการด้านความปลอดภัยที่เข้มงวด แต่ไม่มีระบบใดที่ปลอดภัย 100% ผู้ใช้ควรปฏิบัติตามคำแนะนำดังต่อไปนี้:

  • ตรวจสอบ URL ของเว็บไซต์ก่อนเข้าสู่ระบบ เพื่อป้องกันเว็บไซต์ปลอม
  • หลีกเลี่ยงการดาวน์โหลดโปรแกรมจากแหล่งที่ไม่น่าเชื่อถือ
  • ติดตั้งและอัปเดตโปรแกรมป้องกันไวรัสอย่างสม่ำเสมอ
  • ติดตั้ง Personal Firewall เพื่อป้องกันการโจมตีจากผู้ไม่หวังดี

Website Security Policy (English)

The War Veterans Organization of Thailand recognizes the importance of protecting user data from unauthorized access, disclosure, or destruction.

1. Security Measures

We use Secure Socket Layer (SSL) technology with 128-bit encryption to secure all data transmissions. Users can verify this by checking for "https://" in the website URL.

2. Personal Data Protection

  • Firewall: Restricts access to authorized users only
  • Virus Protection: Updated antivirus systems on all servers and devices
  • Cookies: Used only when necessary, with respect to user privacy
  • Auto Log Off: Automatic session timeout for user safety

3. User Recommendations

  • Verify website URLs before logging in
  • Avoid downloading programs from untrusted sources
  • Install and update antivirus software regularly
  • Use a personal firewall to protect your device
